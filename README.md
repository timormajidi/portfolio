# Portfolio

A professional portfolio website showcasing engineering projects, craftsmanship, and design work.

## Folder Structure

```
portfolio/
├── index.html          # Main portfolio webpage
├── images/            # Project images
├── files/             # Downloadable documents (CV, PDFs, etc.)
└── README.md          # This file
```

## Getting Started

1. **Add images** to the `images/` folder:
   - `main.png` - Hero section image
   - `fs.jpg` - Formula Student project image
   - `peak.jpg` - P.E.A.K. Cyberdeck project image
   - `housing.jpg` - Camera housing project image
   - `trilogy.png` - Three Gifts project image
   - `expbuild.png` - Experimental Builds project image

2. **Add documents** to the `files/` folder:
   - `Timur_MN_CV.pdf` - CV file

3. **Deploy on GitHub Pages**:
   - Go to repository Settings → Pages
   - Select "Deploy from a branch"
   - Choose `main` branch
   - Your site will be live at `https://timormajidi.github.io/portfolio/`

## Features

- **Responsive design** - Works on desktop, tablet, and mobile
- **Smooth scroll animations** - Fade-in effects on page load
- **Modern typography** - DM Sans and DM Serif Display fonts
- **Professional color scheme** - Green, off-white, and dark palette
- **Project showcase** - Grid layout with hover effects
- **Contact section** - Email, LinkedIn, and CV download

- Safari (latest)
